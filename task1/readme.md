# Анализ музыкального стримингового сервиса mosMusic

## Варианты запуска

### Вариант 1. Python-скрипт

#### Требования

* Python 3.13.0

Совместимость с другими версиями Python не проверялась.

#### Запуск

```bash
python main.py
```

Скрипт автоматически подключается к базе данных и выводит результаты всех запросов в консоль.

---

### Вариант 2. Выполнение SQL-скрипта

#### Требования

* Любой инструмент для работы с SQLite (например, DBeaver)

#### Запуск

1. Подключить файл `mosmusic_final_collabs_multiple_albums.db`.
2. Выполнить необходимые запросы из файла `analysis.sql`.

---

## Структура проекта

* `main.py` — выполнение запросов через Python и вывод результатов.
* `analysis.sql` — набор SQL-запросов для анализа данных.
* `mosmusic_final_collabs_multiple_albums.db` — база данных.
